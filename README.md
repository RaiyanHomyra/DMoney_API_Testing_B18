# Project Name: DMoney API Testing

## Project Description:
This project is a Postman-based API testing collection for the DMoney application, covering user creation, activation, and various financial transactions.

## Technology Used:
* NodeJS
* Postman
* Newman
* HTML Report Extra

## How to Run:
* ``npm init -y``
* ``node report.js``

## Test Scenarios:
1. **User Creation** — Create 2 customers and 1 agent using the API
2. **User Activation** — Activate all created users via admin API
3. **System to Agent Deposit** — Deposit 5000 TK from system account to agent account
4. **Agent to Customer Deposit** — Agent deposits 2000 TK to a customer account
5. **Customer to Customer Send Money** — Customer sends 1000 TK to another customer
6. **Customer Cashout** — Another customer cashouts 500 TK from the agent
